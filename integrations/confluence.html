Introduction

Inside the Confluence administration, access the User Macros screen to create the following macros described below.

Value Charts Macro

  • Macro Name: value_charts
  • Macro Title: Value Charts
  • Description: Renders a chart from the VALUE tool's dashboard
  • Categories: Development
  • Macro Body Processing: No macro body
## Macro title: Value Charts
## Macro has a body: N
##
## Developed by: Vitor Freitas
## Date created: 05/05/2017
## Date updated: 23/08/2017
## Installed by: INSTALLER_NAME

## @param ChartType:title=Chart Type|type=enum|enumValues=decision_items_overview,value_ranking,factors_comparison|default=red|desc=Choose a chart type
## @param IssueId:title=Issue ID|type=string|desc=JIRA Issue ID
## @param MeetingId:title=Meeting ID|type=string|desc=VALUE Meeting ID
## @param Width:title=Width|type=string|default=500|desc=Width of the chart
## @param Height:title=Height|type=string|default=400|desc=Height of the chart

<iframe style="border:0;padding:0;margin:0;"
width="$paramWidth" height="$paramHeight"
src="https://VALUE_TOOL_DOMAIN/api/charts/?id=$paramIssueId&meetingId=$paramMeetingId&chartType=$paramChartType">
</iframe>

Change the INSTALLER_NAME and VALUE_TOOL_DOMAIN markups with the correct values.

Value Summary Macro

  • Macro Name: value_summary
  • Macro Title: Value Summary
  • Description: Display the evaluation summary from the VALUE tool
  • Categories: Development
  • Macro Body Processing: No macro body
## Macro title: Value Summary
## Macro has a body: N
##
## Developed by: Vitor Freitas
## Date created: 05/05/2017
## Date updated: 23/08/2017
## Installed by: INSTALLER_NAME

## This is an example macro
## @param IssueId:title=IssueId|type=string|required=true|desc=JIRA Issue ID
## @param MeetingId:title=Meeting ID|type=string|desc=VALUE Meeting ID

<iframe style="border:0;padding:0;margin:0;"
width="100%" height="20"
src="https://VALUE_TOOL_DOMAIN/api/summary/?id=$paramIssueId&meetingId=$paramMeetingId"></iframe>

Change the INSTALLER_NAME and VALUE_TOOL_DOMAIN markups with the correct values.
